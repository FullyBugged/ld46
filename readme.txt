-------------------------------------------------
-              Railroad Lantern                 -
-------------------------------------------------

A Ludum Dare 46 GameJam by Aumniez & Iarwain.

Can be played in solo (high score mode)
or multiplayer (last lantern standing).

Frank Brizuela
Romain Killian

-------------------------------------------------

Controls:

Player1
-------
Move                  - Arrow keys
Dash                  - Right Shift
            .. or ..
Gamepad #1

Player2
-------
Move                  - WASD
Dash                  - Left Shift
            .. or ..
Gamepad #2

Player3
-------
Gamepad #3

Player4
-------
Gamepad #4

Menu
----
Key 1, 2, 3 or 4        - Select Player(s) Count
            .. or ..
A on Gamepad #1         - 1P
A on Gamepad #2         - 2P
A on Gamepad #3         - 3P
A on Gamepad #4         - 4P

Game Over
---------
Space / A               - Restart (game over)
F1 / Back               - Go to Player Select

Misc
----
Space / Start           - Pause (in-game)
F12 / Left Bumper       - Capture a screenshot
